# Genotype Data Folder

This folder currently contains lightweight CSVs used for mapping/ID alignment demos.

You can add full marker panels here for production GWAS:

- HapMap: `*.hmp`
- VCF: `*.vcf` / `*.vcf.gz`
- Marker matrix CSV: first column `sample_id` (or `filename`), remaining columns marker dosages

After copying files here, open **Genotyping / Mapping** in the UI and ingest the selected marker file.
